# Documentacion

Este directorio centraliza documentacion operativa y tecnica del proyecto.

## Estructura

- `mejoras/`: solicitudes, hallazgos y propuestas de mejora registradas por fecha.

## Criterio de registro

Cada nueva solicitud o planteamiento de mejora debe guardarse en `docs/mejoras/` en un archivo Markdown con fecha en el nombre.

Formato sugerido:

- `YYYY-MM-DD-tema.md`

Ejemplo:

- `2026-03-15-mejoras-iniciales.md`
